# Release Notes

Welcome to the LightstonePlatform.Products release notes. Below you'll find detailed information about each release.

## About

LightstonePlatform.Products is a library intended to allow for the surfacing of your products on the Lightstone platform. This package provides you with the required .NET classes to be able to respond to platform requests.
